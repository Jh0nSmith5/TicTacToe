## Descripción

Juego de Tres en Raya para dos jugadores en la terminal, hecho en Dart. Los jugadores se turnan eligiendo una posición del 1 al 9 en el tablero. Gana el primero en alinear tres fichas seguidas (fila, columna o diagonal). Si el tablero se llena sin ganador, es empate. Al terminar cada partida, puedes elegir jugar de nuevo o salir.

---

## Cómo ejecutar

1. Asegúrate de tener [Dart SDK](https://dart.dev/get-dart) instalado.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta el siguiente comando:

```bash
dart run TresEnRaya.dart
```
